# 1
В рамках лабораторной работы был выполнен анализ логов веб-сервера Nginx.

Сначала с помощью команд awk, sort, uniq и head из классического лога access.log были извлечены 10 IP-адресов с наибольшим количеством запросов.

Затем из лога в формате JSON (access_json.log) с использованием jq и аналогичной цепочки команд были также получены топ-10 IP по частоте запросов с подсчётом числа обращений.

После завершения анализа логи были удалены с помощью sudo rm -f.
